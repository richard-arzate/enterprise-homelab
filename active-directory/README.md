# Active Directory

This section documents the Windows Server 2022 Active Directory environment configured within the homelab.

The Active Directory environment was implemented to simulate enterprise identity and access management workflows, including:
- Organizational Unit (OU) structure design
- User and group administration
- Security group management
- Access control
- Group Policy configuration
- DNS management
- Administrative delegation concepts

## Technologies Used
- Windows Server 2022
- Active Directory Domain Services (AD DS)
- DNS
- Group Policy Management
