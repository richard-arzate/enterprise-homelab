# Virtualization

This section documents the Proxmox VE virtualization infrastructure used to host the homelab environment.

Virtualization was implemented to simulate enterprise systems administration, endpoint management, and security testing within isolated environments.

## Virtual Machines
- Windows Server 2022 (DC01)
- Windows 11 Pro Client VM
- Windows 11 Pro Main Workstation VM
- Kali Linux VM

## Technologies Used
- Proxmox VE
- VLAN-Aware Virtual Networking
- Windows Server
- Windows 11 Pro
- Kali Linux

## Objectives
- Simulate enterprise infrastructure
- Practice systems administration
- Create isolated testing environments
- Support cybersecurity training and vulnerability assessment
