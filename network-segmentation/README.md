# Network Segmentation

This section documents the VLAN architecture and network segmentation strategy used throughout the homelab environment.

The environment was segmented into multiple VLANs to improve:
- Security
- Network isolation
- Administrative control
- Device separation
- Traffic management

## Configured Networks
- Management Network
- Trusted Network
- Work Network
- Guest Network
- IoT Network
- Lab Network
- Gaming Network
- Printer Network

## Technologies Used
- UniFi Networking
- VLAN Configuration
- DHCP
- DNS
- Firewall Segmentation Policies

## Objectives
- Separate trusted and untrusted devices
- Isolate lab environments
- Restrict lateral movement
- Improve security posture
